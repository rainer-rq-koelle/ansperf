# ansperf

<!-- badges: start -->

<!-- badges: end -->

The {ansperf} package provides supporting functions for the operational
anlaysis of air transportation / air navigation. It accompanies the
book.

This package is EXPERIMENTAL, i.e. the project started recently and
“things” may change while the ideas and functions firm up. Please make
sure to check for the most recent update/development version.

## Installation

You cannot yet install a released version of ansperf from CRAN

<!-- [CRAN](https://CRAN.R-project.org) with:

``` r
install.packages("ansperf")
```
-->

The development version is available from [GitHub](https://github.com/)
with:

``` r
# install.packages("devtools")
devtools::install_github("rainer-rq-koelle/ansperf")
```
